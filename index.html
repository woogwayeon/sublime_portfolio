DOM에 대해 이해하고 코드를 제작하며 관련 issue에 대응합니다.

innerText, innerHTML를 이용해 사용자에게 입력받은 값을 출력하거나 event와 연계하여 각종 UI를 구현합니다.

addEventListener를 HTML / CSS 연동하여 GNB, LNB, modal 등 동작을 제작합니다.

for, forEach등 반복 명령을 사용해 효율적인 로직을 세우고, 반복작업을 단순화할 수 있습니다.

if, swich문을 사용하여 특정 상황 및 변수를 다양한 상황에서 비교 및 활용하고 코드 제어에 활용합니다.

object(객체형), Array(자료형), map function을 사용하여 효율적인 Card Layout 구현이 가능합니다.

Swiper, AOS, Masony 등 JS plugin 활용하여 웹 페이지 동작 생성이 가능합니다.

Svg와 CSS animation을 이용, getTotalLength Method를 통해 선/면의 흐름 모션을 구현합니다.

if문과 cloneNode, appendChild Method를 사용하여 Gallery 및 carousel 제작이 가능합니다.

scrollY, scrollX property를 scroll event와 구성하여 scroll시 motion을 생성합니다.

IntersectionObserver Object를 이용해서 뷰포트 상태 변화에 따른 scroll motion을 제작합니다.

load 이벤트 핸들러를 이용한 로딩페이지 구현과 부모창과 자식창 사이 명령 제어가 가능합니다.

resize event를 이용하여 반응형 웹 제작 시 JavaScript코드 및 HTML,CSS를 조율합니다.

jQuery 라이브러리 구문 이용하여 간단한 로직을 구현할 수 있습니다.
